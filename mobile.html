📱 iPhone 访问方案

🏠 方案一：局域网（家中/办公室）

  1. 在电脑上以 管理员身份 打开 PowerShell
  2. 运行命令：python -m http.server 8888 --directory "C:\Users\Administrator\Documents\基金监控及推荐\dashboard"
  3. 查看电脑IP：ipconfig → 找到 IPv4 地址（如 192.168.1.100）
  4. iPhone 与电脑连同一WiFi，Safari 打开：http://192.168.1.100:8888/mobile.html
  5. 点 Safari 分享按钮 → 「添加到主屏幕」 即可像App一样使用

💡 添加到主屏幕后，图标显示为"巅峰资本"，点开全屏显示，体验和原生App一样流畅。

🌐 方案二：公网随时访问（推荐）

  1. 安装 Cloudflare Tunnel（免费）或 ngrok
  2. 一行命令：cloudflared tunnel --url http://localhost:8888
  3. 获得公网链接如：https://xxxx.trycloudflare.com
  4. iPhone 直接访问该链接，无需在同一网络
  5. 出门在外、4G/5G 均可实时查看

☁️ 方案三：云部署（永久在线）

  1. 将 dashboard/ 文件夹上传至 GitHub
  2. 开启 GitHub Pages（免费，自动HTTPS）
  3. 获得永久链接如：https://yourname.github.io/apex-capital/mobile.html
  4. 无需开电脑，7×24小时随时访问
